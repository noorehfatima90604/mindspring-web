# MindSpring AI - Frontend 🧠💻

This is the frontend and mobile application repository for **MindSpring AI**, an AI-powered, adaptive tutoring platform tailored for Matric and Intermediate students. This repository contains the complete user interface, adaptive learning flows, and mobile shell integration.

---

## 🛠️ Tech Stack

- **Framework:** React.js (with Vite) for a fast, component-based single-page application.
- **Language:** JavaScript.
- **Mobile Shell:** Capacitor (to package the web application for cross-platform mobile devices).
- **Styling:** [e.g., Tailwind CSS / Bootstrap / Vanilla CSS ]

---

## ✨ Key UI Features Implemented

- **Curriculum-Specific Dashboards:** Custom interfaces for Computer Science, Physics, and Biology students.
- **AI Tutoring Interface:** Interactive chat and learning screens.
- **Active Recall Modules:** Interactive Flashcards and MCQ practice screens.
- **Document Preview:** A4-sized responsive layout preview for generated summaries and quizzes.
